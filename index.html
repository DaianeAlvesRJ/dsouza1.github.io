Tutorial de criação do GitHub Pages

Passo 1: Criar um repositório com o seu nome de usuário e a extensão: .github.io

Exemplo: username.github.io

Passo 2: Clonar repositório

git clone https://github.com/username/username.github.io

Passo 3: Escrevendo na tela "Hello World"

cd username.github.io

echo "Hello World">index.html

Passo 4: Push it

git add --all

git commit -m "Initial commit"

git push -u origin master

Passo 5: http://username.github.io
